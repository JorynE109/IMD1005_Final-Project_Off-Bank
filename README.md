# Off-Bank

Off-Bank is a music blog focusing on showcasing local artists and venues. We also provide a centralized place for people to find shows in the Ottawa area. Our target audiences are young adults in Ottawa looking for alternative music events, and local bands who are looking to share their music and gain attendance at their shows.

## Site Navigation
The site features a side nav bar, which when hovered over expands to show all of the pages.
<br>The **Home** page features an artist highlight and previews of recent articles, clicking these will lead to the Posts page. It also features an upcoming event, clicking this leads to the Events page for more information.
<br>The **Posts** page features all blog posts, which can be sorted by the categories in the filter bar. The search bar at the top also allows users to search for keywords or the title of a specific article.
<br>**Events** features a list of shows by date across a variety of venues. Clicking on the day expands into a more detailed description of the event, such as date, time, venue, and cost. It also features artist biographies and a link to their socials. The list view can be alternated for a calendar view by using the two buttons at the top of the page.
<br>The **About** page has a brief description of our organization and our mission, as well as a contact form for users to provide feeback, submit new artists they would like to see featured, etc.
<br>
In addition to these main pages, any link to an artist, venue, album or event will take you to a page that is dynamically rendered basedon the content you are attempting to access. Thus, an example of a possible user path could look like the following:
1. Home -> Artist highlight
2. User checks out upcoming shows linked on artist page, sees one they are interested in -> event page
3. User looks at all the artists featured in the upcoming show, and wants to learn more -> artist page
This navigation allows a user to follow paths for a long time before having to return to any main site pages.
